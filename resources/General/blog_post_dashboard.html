Introducción

En el marco del programa de Doctorado en Inteligencia Artificial y Aprendizaje Automático, fruto del convenio de colaboración entre la Universidad Autónoma de Santo Domingo (UASD) y la Universidad Miguel Hernández (UMH), hemos desarrollado una herramienta de Learning Analytics orientada a la exploración de datos académicos.

Este proyecto surge como una tarea práctica dentro de la asignatura impartida por el Dr. Óscar Martínez Bonastre, con el objetivo de demostrar cómo las tecnologías open-source y el análisis de datos pueden facilitar la comprensión de fenómenos educativos complejos de manera accesible.

El Objetivo: De los Datos a la Visualización

El propósito principal de este ejercicio fue realizar un Análisis Exploratorio de Datos (EDA) sobre un dataset educativo. En esta primera fase de estadística descriptiva, buscábamos responder preguntas clave mediante la visualización:

  • ¿Cómo se distribuyen los estudiantes por campus y titulación?
  • ¿Existe una correlación visible entre la nota de acceso y el rendimiento universitario?
  • ¿Podemos identificar patrones tempranos de deserción estudiantil?

La Herramienta: Dashboard Interactivo

En el ámbito del análisis de datos existen múltiples herramientas poderosas para la visualización. A continuación, un breve contraste de sus ventajas y desventajas en el contexto académico:

Plataforma Ventajas Principales Desventajas / Limitaciones
Excel Omnipresente, curva de aprendizaje baja, ideal para datos pequeños y análisis rápidos. No escala bien con grandes volúmenes (>1M filas), visualizaciones limitadas, difícil reproducibilidad.
Power BI / Tableau Potentes gráficos "drag & drop", conectores nativos a bases de datos, estándar en Business Intelligence. Costos de licencia elevados para publicación, "cajas negras" (poca flexibilidad en lógica personalizada).
Shiny (R) Excelente para estadística avanzada y gráficos académicos (ggplot2), prototipado rápido. Requiere saber R (menos común en industria general), despliegue a veces complejo.
Python (Dash)
Opción elegida		 Flexibilidad total (todo es código), integración directa con Machine Learning (Scikit-Learn, PyTorch), 100% Open Source. Curva de aprendizaje inicial mayor (HTML/CSS/Callbacks), tiempo de desarrollo más largo que herramientas visuales.

Para materializar este análisis, construimos un dashboard interactivo utilizando:

  • Python (Pandas): Para el procesamiento y limpieza de datos.
  • Dash (by Plotly): Para crear una interfaz web dinámica y visualizaciones interactivas.

Esta combinación permite no solo presentar gráficos estáticos, sino ofrecer una herramienta donde el usuario puede filtrar y "conversar" con los datos en tiempo real, facilitando el descubrimiento de insights.
