🎯 Política de Licencias

MATUASD utiliza un sistema de licencia dual que diferencia entre el código del sitio web y el contenido educativo.

💻 Código del Sitio Web - Licencia MIT

El código fuente del sitio web (HTML, CSS, JavaScript) está licenciado bajo la Licencia MIT.

Esto significa que puedes:

  • Usar, copiar, modificar y distribuir el código
  • Usar el código en proyectos comerciales o privados
  • Incluir el código en software propietario

Única condición: Debes incluir el aviso de copyright original y el texto completo de la licencia MIT en todas las copias o porciones sustanciales del software.

📄 Para más información, consulta el archivo LICENSE en el repositorio del proyecto.

📚 Contenido Educativo - Creative Commons

Todo el contenido educativo disponible en este portal (PDFs, artículos del blog, infografías, materiales didácticos) está licenciado bajo Creative Commons Attribution-NonCommercial-ShareAlike 4.0 International (CC BY-NC-SA 4.0).

Esto significa que puedes:

  • Compartir: Copiar y redistribuir el material en cualquier medio o formato
  • Adaptar: Remezclar, transformar y construir sobre el material

Bajo las siguientes condiciones:

  • Atribución (BY): Debes dar crédito apropiado, proporcionar un enlace a la licencia e indicar si se hicieron cambios
  • No Comercial (NC): No puedes usar el material para propósitos comerciales
